# My Blog - [Backend]

Um blog desenvolvido com **Go + Gin** no backend e **HTML/CSS/JS** no frontend, servindo como plataforma para compartilhar projetos, estudos e documentar minha jornada de desenvolvimento.
 
---

## Sobre o Projeto
 
**My Blog** é um projeto pessoal que funciona como um diário de desenvolvimentos e estudos. A ideia é ter um espaço onde posso:
 
-  Documentar meus projetos e aprendizados
-  Compartilhar meu raciocínio e dificuldades enfrentadas
-  Praticar full-stack development de forma prática
-  Experimentar novos conceitos e tecnologias
-  Servir como portfólio além do GitHub

Este é meu **playground** para arriscar, aprender e reproduzir projetos interessantes à minha forma.

## Arquitetura
 
### Backend
- **Linguagem**: Go
- **Framework**: Gin (HTTP Web Framework)
- **Responsabilidades**:
  - API REST para gerenciar posts
  - Servir páginas HTML (Server-Side Rendering)
  - Gerenciar banco de dados
  - Bucket de requisições
### Frontend
- **HTML**: Estrutura semântica
- **CSS**: Estilização minimalista (terminal-like)
- **JavaScript**: Interatividade e preview em tempo real

---

## O Que Aprendi
 
-  Padrão de Server-Side Rendering com Go
-  Design responsivo minimalista
-  Markdown parsing e rendering no cliente
-  Comunicação entre frontend e backend via APIs REST
-  Gerenciamento de estado em aplicações web
